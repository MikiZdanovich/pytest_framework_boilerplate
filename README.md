# ComplexUiDemo

## This repository contains basic example of usage PageObject pattern with Selenium and Python (PyTest + Selenium).



How To Run Tests
----------------

1) Install all requirements:

    ```bash
    pip3 install -r requirements
    ```
    
2) Run tests:

 ```bash
    python3 -m pytest -v --driver chrome  ~/tests/*
    
OR
    
    Via Pycharm
